food ordering app.

UI/UX Design | User Testing.

For the course "Produktutveckling i medieteknik med metoden "Design-Build-Test" I and three other students were tasked with creating a new design for a pre-existing food delivery application along with a small implementation for how a restaurant could set their delivery area. We managed to get this project through the company named Thnx Innovation which we contacted during the beginning of the course.

In order to create a user friendly and fullfilling design we started by doing some preliminary research into user habits and patterns when it came to the usage of other food ordering applications. After that we began designing the Lo-Fi prototype and continously evaluated and iterated it with the help of user testing.

After creating the final iteration of the Lo-Fi prototype which is pictured above we began designing the Hi-Fi prototype. In the same fashion as the Lo-Fi prototype we continously evaluated and iterated the prototype with the help of user testing.

As of now the project is still ongoing and the implementation has not been finished yet. When it is this section will be updated with information about how we created the tool as well as images of the finished product.
